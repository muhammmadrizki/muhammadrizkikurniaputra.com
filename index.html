SportStore

SportStore is a full-stack e-commerce web application that allows users to browse products, manage their shopping cart, and perform secure authentication.

This project demonstrates a complete integration between frontend, backend, and database in a production-ready environment.

Key Features

  • Product listing and detail pages (dynamic routing)
  • User authentication (register, login, logout, auth me)
  • Shopping cart (add, update quantity, remove items)
  • User dashboard
  • REST API with validation and documentation
  • Fullstack deployment (Vercel, Render, Neon, Cloudflare)
